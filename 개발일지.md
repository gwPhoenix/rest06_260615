#### 1. GitHub Pages 배포 문제 진단 및 수정

**현상**
- `gwphoenix.github.io/rest06_260615/` 접속 시 "CODEWAVE" 텍스트만 표시
- `assets/index-*.js`, `assets/index-*.css` 파일 HTTP 404

**원인**
- 기존 workflow가 `peaceiris/actions-gh-pages@v4`를 사용해 gh-pages 브랜치에 push하는 방식
- GitHub Pages source가 **"GitHub Actions"** 모드로 설정된 경우, 브랜치 내용은 무시되고 Actions 아티팩트만 서빙됨
- 결과적으로 gh-pages 브랜치에 파일이 존재해도 CDN에서 404 반환

**수정**
`.github/workflows/deploy.yml` 을 GitHub 공식 Pages 배포 방식으로 전환

```yaml
# 변경 전
- uses: peaceiris/actions-gh-pages@v4
  with:
    github_token: ${{ secrets.GITHUB_TOKEN }}
    publish_dir: ./dist

# 변경 후
- uses: actions/upload-pages-artifact@v3
  with:
    path: ./dist

- uses: actions/deploy-pages@v4
```

permissions도 `contents: write` → `pages: write` + `id-token: write` 로 변경

#### 2. 라이트모드 nav 메뉴·테마 버튼 미표시 버그 수정

**현상**
- 라이트모드에서 nav 메뉴 링크(강좌, 강사진, 수강후기, 회사소개)가 보이지 않음
- 라이트/다크 모드 토글 버튼도 보이지 않음
- 다크모드에서는 정상 표시

**원인**

`Nav.jsx`의 `on-dark` 클래스 로직:
```js
const onDark = isHome && !scrolled  // 홈 페이지 + 미스크롤 시 적용
```

`on-dark` 클래스가 적용되면 nav 텍스트와 아이콘이 흰색(`#fff`)으로 설정됨.

그런데 `Home.jsx`의 `<main>`에 `paddingTop: 'var(--nav-h)'`(76px)가 있어 히어로 섹션이 nav 아래에서 시작. nav 뒤에는 `<main>`의 빈 패딩 영역이 노출되는데:

| 모드 | body 배경(`--bg`) | nav 뒤 색상 | nav 텍스트 | 결과 |
|------|-----------------|------------|-----------|------|
| 라이트 | `#FFFFFF` (흰색) | 흰색 | 흰색 | **invisible** |
| 다크 | `#0A0E1C` (어두움) | 어두운색 | 흰색 | 정상 표시 |

**수정**
`Home.jsx`에서 `<main>`의 paddingTop을 제거하고, 히어로 섹션이 nav 영역까지 덮도록 수정

```jsx
// 변경 전
<main style={{ paddingTop: 'var(--nav-h)' }}>
  <section style={{ ..., paddingTop: 80, ... }}>

// 변경 후
<main>
  <section style={{ ..., paddingTop: 'calc(var(--nav-h) + 80px)', ... }}>
```

히어로 섹션의 어두운 배경(`var(--ink)`)이 nav 영역(y=0~76px)까지 커버하게 되어 라이트모드에서도 흰 nav 텍스트가 정상 표시됨.

#### 4. ChatWidget 전체 페이지 전역 적용

**현상**
- `ChatWidget`이 `Home.jsx`에만 있어 About, Courses, Course, Checkout, Instructors, Reviews, Profile 페이지에서 챗봇 미표시

**수정**
`Home.jsx`에서 제거하고 `App.jsx`에 전역으로 이동

```jsx
// App.jsx
import ChatWidget from './components/ChatWidget'

export default function App() {
  return (
    <AuthProvider>
      <HashRouter>
        ...
        <Footer />
        <ChatWidget />
      </HashRouter>
    </AuthProvider>
  )
}
```

**결과**
- 사이트 전체 페이지 우측 하단에 챗봇 플로팅 버튼 표시

#### 5. 챗봇 풍선도움말 상시 표시 + float-soft 애니메이션

**내용**
- 채팅이 닫혀 있는 동안 말풍선 **항상 노출** (지연·닫기 로직 없음)
- 말풍선 문구: "💬 AI 학습 도우미 — 특강 일정·n8n·AI 영상 등 궁금한 건 여기서 물어보세요!" + **채팅 시작 ↗** 버튼
- 말풍선 + FAB를 `chat-float-cluster`로 묶어 함께 위아래로 float
- `@keyframes float-soft` (±10px, 4초 주기) 적용
- 채팅 팝업 열리면 float 중지 (`floating` 클래스 조건부 적용)

```css
@keyframes float-soft {
  0%, 100% { transform: translateY(0); }
  50%       { transform: translateY(-10px); }
}
.chat-float-cluster.floating { animation: float-soft 4s ease-in-out infinite; }
```

**결과**
- 모든 페이지 우측 하단에 말풍선이 항상 표시되며 부드럽게 위아래로 움직임
- 채팅 팝업과 분리되어 팝업 열릴 때 float 중지

#### 6. 메인 히어로 텍스트 크기 축소

**현상**
- "코드로 세상을 바꿔라." 텍스트가 지나치게 크게 표시됨

**수정**
`.h-xxl` 클래스 폰트 사이즈 축소

```css
/* 변경 전 */
.h-xxl { font-size: clamp(48px, 9vw, 140px); }

/* 변경 후 */
.h-xxl { font-size: clamp(38px, 6.5vw, 104px); }
```
